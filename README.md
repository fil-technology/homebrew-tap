# fil-technology/homebrew-tap

Homebrew tap for Fil Technology command-line tools.

## Install Ashex

```bash
brew install fil-technology/tap/ashex
```

You can also tap first and then install:

```bash
brew tap fil-technology/tap
brew install ashex
```

## Install Esh

```bash
brew tap fil-technology/tap
brew install --cask esh
```
